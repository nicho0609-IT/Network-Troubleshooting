# Network-Troubleshooting-Labs

## Overview

This repository contains a collection of network troubleshooting labs designed to practice diagnosing common network connectivity issues using basic networking tools.

The purpose of these labs is to simulate real-world troubleshooting scenarios .

Each lab focuses on identifying the root cause of a specific networking issue and verifying the resolution using standard diagnostic tools.

## Lab Environment

All labs in **this repository** were conducted in a home lab environment using a single computer and a standard home network.
The goal is to demonstrate that many fundamental networking troubleshooting skills can be practiced without complex infrastructure.

## Hardware Used

These labs were performed using minimal hardware:

- Personal computer (Windows OS)
- Home router
- Standard home internet connection

No enterprise networking equipment was required.

## Lab Methodology

Each lab follows a consistent troubleshooting workflow:

1. Identify the problem (Ticket)
2. Investigate using diagnostic tools
3. Analyze the results
4. Determine the root cause
5. Apply the resolution
6. Verify that the issue is resolved
